# cb_dx_pre

---

This little script will help you find out what the spread of the CB frequency has been like in the past few days.

---
### First 

First make a link to local bin folder, and use it. 

```bash
This data is the past
λ> git clone https://github.com/zsoltibaba37/cb_dx_pre.git
λ> sudo ln -s /home/USERNAME/cb_dx_pre/cb_check.py /usr/local/bin/dxcheck  
    or whatever you want
λ> dxcheck
```
---

```bash
This data is current
λ> python dxcheck.py

--- 11m / CB Current Data ---
Solar Flux (SFI): 112
K-index:          2.0
A-index:          40
Propagation:      GOOD
---------------------------------
```
